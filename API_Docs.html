ClientsList

Up 
get /clients
Client's list (clientsGet)
The clients endpoint returns the client's list and their providers.

Return type

array[Client]

Example data

Content-Type: application/json
[ {
  "phone" : "aeiou",
  "name" : "aeiou",
  "client_providers" : [ {
    "id" : ""
  } ],
  "id" : "",
  "email" : "aeiou"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An array of clients
